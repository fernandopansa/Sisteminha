# Sisteminha
ERP para micro e pequenas empresas

O Objetivo inicial é criar um sistema com:

- Produtos
    - Adicionar Produtos
    - Listar Produtos
    - Buscar Produtos
    - Relatórios / Produtos
- Serviços
    - Novo Serviço
    - Serviços em Andamento
    - Serviços Concluídos
    - Relatórios / Serviços
- Vendas
    - Realizar vendas
    - Consultar vendas
    - Relatórios / Vendas
- Clientes
    - Cadastrar Clientes
    - Listar Clientes
    - Buscar Clientes
    - Relatórios / Clientes
- Fornecedores
    - Cadastrar Fornecedor
    - Listar Fornecedores
    - Buscar Fornecedores
    - Relatórios / Fornecedores 
- Trocas (RMA)
